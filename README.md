cliente (admin)\n- `POST /api/make-admin` - Promover admin (admin)\n- `POST /api/create-user` - Criar usuário (admin)\n\n### Rotas\n- `POST /api/calculate-route` - Calcular rota otimizada\n- `POST /api/gh-calculate-route` - Rota via GraphHopper\n- `POST /api/pois-for-route` - Buscar POIs na rota\n- `POST /api/validate-route` - Validar violações\n\n### Utilidade\n- `GET /api/ping` - Health check\n- `POST /api/contact` - Formulário de contato\n\n---\n\n## 🐛 Troubleshooting\n\n### Servidor não inicia\n\n`bash\n# Verificar se port 8081 está em uso\nGet-NetTCPConnection -LocalPort 8081\n\n#
