Health check\nInvoke-WebRequest -Uri \"http://localhost:8081/api/ping\" | Select-Object -ExpandProperty Content\n\n# Resultado esperado:\n# {\"ok\":true,\"now\":\"2025-12-18T...Z\"}\n\n# 2. Verificar autenticação (sem token)\nInvoke-WebRequest -Uri \"http://localhost:8081/api/auth-status\" | Select-Object -ExpandProperty Content\n\n# Resultado esperado:\n# {\"loggedIn\":false}\n\n# 3. Testar com token inválido\n$headers = @{ Authorization = \"Bearer INVALID_TOKEN\" }\nInvoke-WebRequest -Uri \"http://localhost:8081/api/auth-status\" -Headers $headers | Select-Object -ExpandProperty Content\n`\n\n## Gerar Certificados SSL\n\n`bash\n# Windows PowerShell (com OpenSSL instalado)\ncd back-end\\certs\n\nopenssl req -x509 -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365 -nodes -subj \"/CN=localhost\"\n\n# Ou manualmente (Enter para valores padrão)\nopenssl req -x509 -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365 -nodes\n\n# Verificar certificado\nopenssl x509 -in cert.pem -text -noout\n`\n\n## Limpeza e Manutenção\n\n`bash\n# Limpar node_modules\ncd back-end\nrm -r node_modules\nnpm cache clean --force\nnpm install\n\n# Verificar versão Node\nnode --version\nnpm --version\n\n# Atualizar dependências\nnpm update\n`\n\n## Debugging\n\n`bash\n# 1. Listar processos Node rodando\nGet-Process node\n\n# 2. Matar processo na porta 8081\nGet-NetTCPConnection -LocalPort 8081 | Select-Object -ExpandProperty OwningProcess | Stop-Process -Force\n\n# 3. Verificar se porta está livre\nTest-NetConnection -ComputerName localhost -Port 8081\n\n# 4. Ver logs em tempo real\ncd back-end\nnpm run dev 2>&1 | Tee-Object -FilePath .\\server.log\n`\n\n## Git (se aplicável)\n\n`bash\n# Status\ngit status\n\n# Committar mudanças\ngit add .\ngit commit -m \"Correção: problemas de segurança e middleware\"\n\n# Push\ngit push origin main\n`\n\n## Firebase (CLI)\n\n`bash\n# Instalar CLI\nnpm install -g firebase-tools\n\n# Login\nfirebase login\n\n# Deploy\nfirebase deploy\n\n# Logs em tempo real\nfirebase functions:log\n`\n\n## Docker (Futuro)\n\n`bash\n# Build imagem\ndocker build -t road-truck:latest .\n\n# Rodar container\ndocker run -p 8081:8081 --env-file .env road-truck:latest\n\n# Verificar container\ndocker ps\n\n# Logs\ndocker logs <container_id>\n`\n\n## Testes com Postman/cURL\n\n`bash\n# 1.
